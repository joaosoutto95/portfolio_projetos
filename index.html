Sobre mim

Meu nome é João Pedro Soutto, formei em Engenharia Civil pela UFRJ em 2021. Ao ser efetivado como Engenheiro Júnior e exercer os conhecimentos da minha área de estudo, notei que o meu maior interesse sempre foi a obtenção, limpeza, análise, interpretação e impacto nos resultados dos dados de um negócio, independente do setor estudado. Com o objetivo de aprimorar essas análises, comecei o estudo da Ciência de Dados em 2020. Atualmente trabalho na área de segurança de barragens para uma consultoria de engenharia e mantenho uma rotina intensa de estudos em data science.

Projetos em Ciência de Dados

Previsão de vendas rede de farmácias

Uso de Python, estatística e técnicas supervisionadas de machine learning para elaboração de um algoritmo de previsão de vendas a partir de dados temporais. O objetivo é descubrir insights do negócio e usar as descobertas para escolher e treinar um modelo de previsão. Neste caso, o setor avaliado é uma rede de farmácias, na qual a empresa busca prever o faturamento nas próximas 6 semanas (relativo a data do último dado fornecido). MAPE de 4% obtido pelo modelo de XGBoost Regressor

    Ferramentas utilizadas:

  • Python
  • XGBoost Regressor
  • Flask
